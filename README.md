# 🎬 Dynamic Movie & TV Show Search

------------------------------------------------------------------------

## 🚀 Overview

This project is a **dynamic search application** built with JavaScript
that filters a local dataset of movies and TV shows based on user input.
It focuses on delivering a smooth user experience using efficient array
methods and dynamic DOM updates.

------------------------------------------------------------------------

## ✨ Features

-   🔍 Real-time search using JavaScript array filtering\
-   🎨 Modern and responsive UI design\
-   📦 Dynamic rendering of results (cards layout)\
-   ⚠️ User-friendly message when no results are found\
-   🔘 Interactive buttons with improved styling\
-   🧹 Clean and organized code structure

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   HTML5 -- Structure of the application\
-   CSS3 -- Styling and layout\
-   JavaScript (ES6+) -- Core logic and dynamic behavior\
-   Array methods (`filter`, `map`) -- Data manipulation

------------------------------------------------------------------------

## ⚙️ How It Works

1.  A local array of objects stores movie and TV show data.\
2.  The user types in the search input.\
3.  JavaScript listens for input events.\
4.  The `filter()` method is used to find matching results.\
5.  Results are dynamically rendered as cards in the UI.\
6.  If no matches are found, a custom message is displayed.
